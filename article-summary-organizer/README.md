# 📰 Article Summary Organizer

**Article → AI summary → email delivery**

## Workflow

```
Google Form (Title + Content + Category)
        ↓
    Make.com
        ↓
  OpenAI ChatGPT
        ↓
  Structured JSON
        ↓
  Gmail (auto-send)
```

## JSON Output
```json
{
  "key_takeaways": ["point 1", "point 2", "point 3"],
  "action_items": ["action 1", "action 2"],
  "has_actions": true
}
```

**Tech:** Make.com · OpenAI · Google Forms
