# Lovable.dev Landing Page Prompts

Reusable prompts for generating production-ready landing pages using Lovable.dev.

---

## Prompt 1: AI Tool / SaaS Landing Page

```
Build a modern, conversion-focused landing page for an AI automation tool called [TOOL NAME].

Design requirements:
- Hero section: bold headline, 1-sentence value prop, CTA button ("Start Free Trial")
- Problem/Solution section: 3 pain points with icons, followed by the solution
- Features section: 3 key features with icons and short descriptions
- Social proof: placeholder for 3 testimonials with name, title, company
- Pricing section: 3 tiers (Free, Pro, Enterprise) with feature lists
- Footer: logo, nav links, social icons

Style: Clean, minimal, dark mode, blue/purple gradient accents, Inter font
Framework: React + Tailwind CSS
Make it fully responsive and production-ready.
```

---

## Prompt 2: Service Business Landing Page

```
Create a professional landing page for a [SERVICE TYPE] business called [BUSINESS NAME].

Sections needed:
- Hero: photo background placeholder, headline, subtitle, "Get a Free Quote" CTA
- About Us: 2-3 sentences, trust badges (Licensed, Insured, 5-star rated)
- Services: 4 service cards with icon, title, short description
- Why Choose Us: 3 differentiators (speed, quality, price)
- Contact Form: name, email, phone, message fields
- Google Maps placeholder embed
- Footer with address, phone, social links

Style: Professional, trustworthy, white/navy blue, clean typography
Make it mobile-first and accessible.
```

---

## Prompt 3: Lead Capture / Waitlist Page

```
Build a minimal, high-converting waitlist landing page for [PRODUCT NAME].

Requirements:
- Single-column layout
- Headline: bold problem statement (10 words or less)
- Subheadline: the promise / solution (1 sentence)
- Email capture form with "Join the Waitlist" button
- 3 bullet points: what they'll get when they join
- Social proof counter: "Join 1,200+ people already on the list"
- No navigation, no footer distractions

Style: Dark background, bright accent CTA button, centered content, Framer-motion micro-animations if possible.
```

## Prompt 4: Portfolio / Personal Brand Page

```
Build a personal portfolio landing page for an AI Automation Specialist.

Sections:
- Hero: name, title, 1-line bio, "View My Work" + "Contact Me" CTAs
- Skills grid: icons for Make.com, n8n, Zapier, UiPath, OpenAI, Twilio
- Featured Projects: 3 project cards with title, description, tech tags, and link
- About: short paragraph, photo placeholder
- Contact: email form

Style: Dark mode, minimal, developer aesthetic, monospace accent font for tech terms, smooth scroll.
Tech: React + Tailwind
```

---

## Tips for Lovable.dev Prompting

1. **Be specific about sections** — list every section you need
2. **Specify the style** — color palette, font, dark/light mode
3. **Mention the framework** — React + Tailwind gets the best results
4. **Ask for responsiveness** explicitly or it may skip mobile
5. **Iterate in follow-up prompts** — first prompt = structure, second = style polish
